# 🏓 Pong Game

A classic two-player Pong game built using **Python Turtle Graphics** and **Object-Oriented Programming (OOP)**.

## ✨ Features

- 🏓 Two-player gameplay
- 🎮 Smooth paddle controls
- ⚡ Ball bounces off walls and paddles
- 📊 Live score tracking
- 🚀 Ball speed increases after every paddle hit
- ➖ Center dashed line

## 🛠️ Tech Stack

- Python
- Turtle Graphics
- Object-Oriented Programming (OOP)

## 🎮 Controls

| Player | Keys |
|--------|------|
| Left Paddle | **W / S** |
| Right Paddle | **↑ / ↓** |

## ▶️ Run the Project

```bash
python main.py
```
